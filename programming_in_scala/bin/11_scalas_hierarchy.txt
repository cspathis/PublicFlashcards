How do Scala classes extend AnyRef?<br>What does this provide?<br>	With <tt>ScalaObject</tt>.<br><br>This only provides <tt>$tag</tt>, for pattern matching speed.<br>
What does <tt>AnyRef</tt> add to <tt>java.lang.Object</tt>?<br>	`final eq<br><br>final ne<br><br>final ==<br><br>final !=`<br>
What does <tt>Any</tt> provide?<br>	<tt>final ==</tt> (forwards to equals)<br><tt>final !=</tt> (opposite of ==, guaranteed)<br><br>`equals(Any)<br>hashCode()<br>toString()<br>final isInstanceOf[T]<br>final asInstanceOf[t] : T` (cast)<br>
What does <tt>AnyVal</tt> add?<br>	nothing<br>
How are value types special?<br>	They are declared abstract and final to prevent instantiation with the new keyword.<br>
To what does every value type have implicit conversion?<br>	It has conversion to a rich type in <tt>scala.runtime</tt> that provides more methods.<br>
What is the terminology of the value types hierarchy?<br>	<pre>                            value types<br>                         /        |      \<br>            numeric types      Boolean    Unit<br>               /                  |<br>        integer types        floating point types<br>         /        |   \           |           \<br>subrange types   Int  Long     Float        Double<br>   /    |    \<br>Byte  Short  Char<br></pre>
What is the top of the Scala type hierarchy?<br>	<pre>            Any<br>         /       \<br>      AnyVal   AnyRef<br>               (on JVM = java.lang.Object)<br></pre>
What is the <tt>AnyRef</tt> hierarchy?<br>	<pre>                        AnyRef<br>                   java.lang.Object<br>                 /           |          \<br>     ScalaObject           other Java   java.lang.String<br>     /         \             classes<br>  Iterable   other Scala<br>    |          classes<br>   Seq<br>    |<br>   List<br></pre>
What is a literal identifier?<br>What does it do?<br>	Any sequence of acceptable characters enclosed by back ticks.<br><br>It forces identifier interpretation over keywords or other restrictions.<br><br>E.g., <tt>Thread.\</tt>yield\<tt>()</tt><br>
